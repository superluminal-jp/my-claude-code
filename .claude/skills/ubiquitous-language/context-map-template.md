## 関係種別 凡例

| 種別 | 意味 |
|------|------|
| `same-name-different-meaning` | 同名・異義（コンテキストごとに定義が違う） |
| `synonym` | 異名・同義（同じ概念を異なる名前で呼ぶ） |
| `identical` | 完全同一（コンテキストをまたいで同義・同名） |
| `refines` | 一方の定義が他方を細分化している |
